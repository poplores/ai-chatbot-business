# AI Chatbot for Small Businesses
Custom AI chatbot built with Python and the OpenAI API.
Customizable for any small business in seconds.
Built by a CS student @ UMD.

## Real Client Demo — Hands4YouAssembly
Built a custom chatbot for a local assembly business in the DMV area.
Handles pricing questions, service area inquiries, and directs 
customers to book via phone or email automatically.
